## Implementasi Fitur Workflow

Implementasi fitur: $@

Ikuti workflow berikut secara berurutan:

### Langkah 1: Scout (Eksplorasi)
- Cari kode yang relevan dengan fitur ini (grep, find, ls)
- Pahami struktur yang sudah ada
- Identifikasi file mana yang perlu diubah

### Langkah 2: Plan (Perencanaan)
- Buat rencana implementasi
- Sebutkan file yang akan diubah/dibuat
- Jelaskan approach yang akan digunakan
- Minta persetujuan sebelum lanjut

### Langkah 3: Implementasi
- Implementasi sesuai rencana
- Commit tiap perubahan bermakna
- Pastikan tidak merusak fungsionalitas existing

### Langkah 4: Verifikasi
- Cek apakah fitur berfungsi
- Pastikan tidak ada breaking changes
- Review hasil akhir
